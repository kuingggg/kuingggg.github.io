yes94 发表于 2013-4-20 22:59
$a + b + c = 3$

$a^2  + b^2  + c^2  \ge \frac{(a + b + c)^2 }{3} = 3$

$ab + bc + ca = \frac{(a + b + c)^2  - (a^2  + b^2  + c^2 )}{2} \le 3$

$\frac{1}{a + ab} + \frac{1}{b + bc} + \frac{1}{c + ca} \ge \frac{(1 + 1 + 1)^2 }{a + b + c + ab + bc + ca} \ge \frac{3}{2}$
