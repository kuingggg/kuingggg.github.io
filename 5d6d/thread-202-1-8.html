\[\left( {\frac{{1 + i}}{2}} \right){\left( { - i} \right)^{n + 1}} + \left( {\frac{{1 - i}}{2}} \right){i^{n+1}}\]
\[\cos \frac{{n + 1}}{2}\pi  + \sin \frac{{n + 1}}{2}\pi \]
\[{\left( { - 1} \right)^{\left\lceil {\frac{n}{2}} \right\rceil }}\]
