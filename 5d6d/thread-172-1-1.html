很多选项，帮助里各种例子看得我都

简单的例子：
Animate[Plot3D[Sin[x + a] Sin[y + b], {x, 0, 2 Pi}, {y, 0, 2 Pi}, PlotRange -> 1, PlotPoints -> 50], {a, 0, Pi}, {b, 0, Pi}]

出来的东东可以点击暂停参数，然后可以手动拖动改变参数值观察变化。
