【Firefox】

先装猴子 Greasemonkey ：
https://addons.mozilla.org/zh-CN/firefox/addon/greasemonkey/

再装 Tex the world 脚本：
http://thewe.net/tex/

把公式夹在 [; 和 ;] 之间，即可自动转换；

如果需要关闭转换，点击右上角猴子图标关掉脚本即可。

【Chrome】

Render any code between [; and ;] into LaTeX images.

https://chrome.google.com/websto ... hkkcgdnmfmhhbjmhggn
