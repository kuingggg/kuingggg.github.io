三元素子集中最小元素为$k$的子集个数为$C_{n - k}^2$
且$1 \le k \le n - 2$
所以\[\begin{array}{l}
P_n  = \sum\limits_{k = 1}^{n - 2} {kC_{n - k}^2 }  \\
  = \sum\limits_{k = 1}^{n - 2} {C_{n - k}^2 }  + \sum\limits_{k = 2}^{n - 2} {C_{n - k}^2 }  + \sum\limits_{k = 3}^{n - 2} {C_{n - k}^2 }  +  \ldots  + C_2^2  \\
  = C_n^3  + C_{n - 1}^3  + C_{n - 2}^3  +  \ldots  + C_2^2  \\
  = C_{n + 1}^4  \\
\end{array}\]
