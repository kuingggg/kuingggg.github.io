目测不存在啊，取 $a = b = \frac{1}{2} + \varepsilon $ 则 $\mathop {\lim }\limits_{\varepsilon  \to {0^ + }} c =+\infty $
于是
\[\frac{1}{{1 + a}} + \frac{1}{{1 + b}} + \frac{1}{{1 + c}} \to \frac{4}{3}\]
